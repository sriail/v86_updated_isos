# ISO Directory

Place SliTaz 5.0 Rolling 32-bit ISO in this directory.

## Recommended ISO

**SliTaz 5.0 Rolling 32-bit**
- Download from: http://mirror.slitaz.org/iso/rolling/
- Recommended file: `slitaz-rolling.iso` or `slitaz-rolling-core.iso`
- File should be named: `slitaz-rolling.iso`

## Alternative Method

You can also use the "Import" button in the web interface to load ISO files directly without placing them in this directory.

## Supported Files

- `.iso` - ISO disk images
- `.img` - Raw disk images
- `.state` - Saved emulator states
